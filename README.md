# Portfólio | Victor Rafael Gomes

Portfólio profissional desenvolvido em React, TypeScript e Vite, com suporte a português/inglês e tema claro/escuro.

## Stack

- React
- TypeScript
- Vite
- Lucide React
- CSS moderno com variáveis de tema

## Arquitetura

```text
src/
  assets/       imagens e arquivos visuais
  components/   seções e componentes reutilizáveis da interface
  data/         conteúdo do portfólio em português e inglês
  hooks/        hooks compartilhados
  types/        tipos TypeScript do domínio
```

## Scripts

```bash
npm install
npm run dev
npm run build
npm run lint
```

## Conteúdo

As informações principais ficam em `src/data/portfolio.ts`, facilitando ajustes de stack, experiências, links e textos nos dois idiomas.
